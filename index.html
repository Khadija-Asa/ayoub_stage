03 - Mon troisième jour


Ce mercredi, j'ai pu me renseigner sur différents langages de programmation. J'ai pu connaitre leur utilité, leur avantages et inconvénients. Aujourd'hui,ils y a divers langages de programation dans le monde mais ils n'ont pas tous la même utilité c'est pour ça que je voudrais vous en présenté quelque-un.

html

Ce langage fait parti du b.a ba du développement web. Le HTML sert à structurer la page web et ainsi de constituer son squelette.

css

Ce langage fait parti du b.a ba du développement web. Le CSS sert à styliser en intégrant des éléments visuels et graphiques.

javascript

Ce langage sert à intégré des scrpits géneralement pour ajouter une animation ou un effet visuel. Il permet aussi d'ajouter de l'intéractivité.

github

Github est un site web et un service cloud qui aide les développeurs à stocker et à gérer leur code ainsi qu'à controler les mofications qui lui sont apporté.

python

Ce langage est le script de l'automatisation, il permet d'automatiser les intéraction avec les navigateurs web. Il est sutout utilisé par les informatien spécialicé dans l'IA c'est à dire l'intelligence artificiel.

linux

Linux est un stytème d'exploitation permettant de gérer l'execution d'application. Il se charge également de piloter l'ensemble des processus matériels associé, information. Linux est open-source.
