mon 1er jour


Connaissance des bases
du développement web

J'ai pris connaissance des bases du développement web plus précisément les deux languages de programmations basiques mais essentiel et obligatoire pour la création d'un site. J'ai appris a identifier les différentes sections d'un site web et enfin, j'ai appris à structurer avec HTML et styliser avec CSS un site web.


Durant l'après-midi, j'ai commencé la création de mon site avec les informations et les compétences que j'ai pu apprendre. J'ai commencé avec la landing page c'est à dire la page d'accueil. Puis j'ai définis les couleurs les plus appropriées pour le site, ce qu'on appelle un moodboard. Ensuite, j'ai écris les premières phrases et choisi le background c'est à dire l'arrière-plan.

mon 2ème jour


Ce mardi, j'ai créé et défini mon logo que j'ai ensuite placé sur la page d'accueil. Je me suis aidé de Canva afin de m'inspirer pour le design et également pour le créer et pouvoir l'utiliser sur mon site web.


Puis j'ai stylisé la page. J'ai pu en apprendre davantage sur les langages informatiques. J'ai vu comment faire pour déplacer et positionner des éléments et des formes sur les pages. J'ai appris à régler les tailles, à créer des textes, faire des designs et bien plus encore. Cet après-midi, je me suis exercé sur les bases que j'ai pu apprendre ce début de semaine.

mon 3ème & 4ème jour


Ce mercredi, j'ai pu me renseigner sur différents langages de programmation. J'ai pu connaitre leur utilité, leurs avantages et inconvénients. Aujourd'hui, il y a divers langages de programmation dans le monde mais ils n'ont pas tous la même utilité c'est pour ça que je voudrais vous en présenter quelques-un. Ce jeudi, j'ai donc pu apprendre à créer une grid c'est à dire des tableaux et à placer des éléments dans celles-ci. Puis, j'ai commencé à apprendre a créer des effets visuels et des animations. Cet aprés-midi, j'ai mis en oeuvre ce que j'ai appris en m'exercant sur ce tableau de languages.






html

Ce langage fait parti du b.a ba du développement web. Le HTML sert à structurer la page web et ainsi de constituer son squelette.



css

Ce langage fait parti du b.a ba du développement web. Le CSS sert à styliser en intégrant des éléments visuels et graphiques.



javascript

Ce langage sert à intégré des scrpits géneralement pour ajouter une animation ou un effet visuel. Il permet aussi d'ajouter de l'intéractivité.



github

Github est un site web et un service cloud qui aide les développeurs à stocker et à gérer leur code ainsi qu'à controler les mofications qui lui sont apporté.



python

Ce langage est le script de l'automatisation, il permet d'automatiser les intéraction avec les navigateurs web. Il est sutout utilisé par les informatien spécialicé dans l'IA c'est à dire l'intelligence artificiel.



linux

Linux est un stytème d'exploitation permettant de gérer l'execution d'application. Il se charge également de piloter l'ensemble des processus matériels associé, information. Linux est open-source.
